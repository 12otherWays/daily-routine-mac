Daily Routine

A native macOS daily task & routine tracker. Plan your day, reuse routines as templates, and watch your consistency build up in a GitHub-style contribution heatmap. Local-first — no accounts, no network.

What it does

Features

Everything you need to plan a day and keep a streak going.

🗓️

Day / Week / Month

Track tasks per day, or zoom out to a weekly grid or monthly calendar.

📊

Stats & streaks

All-time stats, current & longest streaks, and a rolling 12-month contribution heatmap.

📋

Templates

Save a set of tasks as a named routine and apply it to any day in one click.

🏷️

Categories & priorities

Tag tasks high / med / low and group them by your own custom categories.

🔐

Export & import

Back up and restore your data as a portable .drbackup file.

💾

Local-first

All data lives in a single JSON file on your Mac. No accounts, no cloud.

Get started

Build from source

Requires Xcode Command Line Tools — full Xcode is not needed to build or run.

# clone & run
git clone https://github.com/12otherWays/daily-routine-mac.git
cd daily-routine-mac
swift run

# or package a distributable .dmg
./scripts/build-dmg.sh   # → build/Daily Routine.dmg
1.Install command line tools: xcode-select --install
2.Run swift run to build and launch the app.
3.Run ./scripts/build-dmg.sh to produce a shareable installer.

Exported backups are obfuscated, not securely encrypted — a deliberate trade-off for a low-sensitivity to-do app. See the README for details.
